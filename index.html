Sangin Snoop

Watch availability checker for Sangin Instruments

Python Auto-discovery Discord alerts

Monitors Sangin Instruments for watch availability. Automatically discovers all products and alerts you via Discord when watches come back in stock.

View on GitHub Fork

What it does

  • Auto-discovers all watches from the Sangin website (currently finds 30+ products)
  • Checks each product page for "Sold Out" or "Add to Cart" status
  • Sends Discord notifications when availability changes
  • Alerts you when new watches are added to the store
  • Runs automatically every 30 minutes via GitHub Actions

Quick start

# Clone and run
git clone https://github.com/arandomguyhere/Sangin-Snoop.git
cd Sangin-Snoop
pip install -r requirements.txt
python Sanginsnoop.py

Setup Discord alerts

  1. Fork this repo
  2. Create a Discord webhook (Server Settings → Integrations → Webhooks)
  3. Add DISCORD_WEBHOOK_URL to your repo's secrets (Settings → Secrets → Actions)
  4. The workflow runs every 30 minutes automatically
